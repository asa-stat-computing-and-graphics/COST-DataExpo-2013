# QuachSymanzikForsgren
This repository contains supplementary materials for the paper by Quach et al. (2019) to enable the reader to reproduce
the numerical analysis and the graphical results presented in that paper.

The full reference is:
Quach, A., Symanzik, J., Forsgren, N. (2019) "Soul of the Community: An Attempt to Assess Attachment to a Community", Computational Statistics 34(4):1565-1589.

The final publication is available at the Springer web site via https://doi.org/10.1007/s00180-019-00866-2. © Springer-Verlag GmbH Germany, part of Springer Nature, 2019.
